About

My name is Anas, and I'm a developer with a strong background in web backend dev ( Vanilla PHP, Laravel framework ) and a good knowledge of SQL and NoSQL databases. I also have experience with front-end development using the HTML/CSS/JS stack, Bootstrap, and with some JS packages like Swiper.js, AOS ... And with SPA development with LiveWire.

In addition to my development skills, I have over four years of experience in SysAdmin, especially on GNU/Linux Administration, which has allowed me to gain expertise in scripting languages like Python and Bash, learn how to set-up tech stacks, and have a good understanding of application layer protocols such as FTP, SSH, NFS or Samba.

I am therefore extremely familiar with command-line interfaces and the tools present within, such as the GNU Coreutils, as well as other development-related tools such as git. I'm also passionate about WebSec, and I'm familiar with many tools and techniques related to that field which allows me to write clean, efficient, and secure code.
